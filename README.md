# i.MX RT Peripheral Access Crates

i.MX RT peripheral access crates and patches

This is very much modeled on the great stm32-rs crate which does much the same
for ST's fantastic STM32 family. 

## Supported Devices

* imxrt1011
* imxrt1015
* imxrt1021
* imxrt1051
* imxrt1052
* imxrt1061
* imxrt1062
* imxrt1064
